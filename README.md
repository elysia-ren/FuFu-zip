# FuFu-zip v1.1.0

安全文件压缩解压工具，支持文件名全加密、AES-256 内容加密。
v1.1.0与v1.0.0对文件名的加密不同，压缩与解压请使用同一版本

## ✨ 特性

### 保护您的文件隐私
FuFu-zip 将文件名和内容全面加密，加密后的文件在任何第三方眼中都只是无法解读的数据，从而保护您的隐私不被窥探。

### 全面隐藏文件类型
加密后所有文件统一显示为 `.enc` 后缀，无论原始格式是什么，
对外都呈现为相同的无特征文件。**文件类型、用途、内容均不可被推测。**

### 加密保护
采用 AES-256 加密算法，即使文件被他人获取，没有正确的密码也无法解密。
压缩包内容与文件名均受到同等强度的保护。

### 零门槛自动解密
您和您的接收方只需安装FuFu-zip，无需交换密码，即可自动解密。
软件内置 100 条高强度随机密码，全程无感，兼顾安全与便利。

## 📦 安装

### 依赖

```bash
pip install pyzipper pycryptodome
```

### 运行

```bash
python zip_tool_modern_v1_0_0.py
```

## 🚀 使用

### 压缩

1. 点击 **添加文件** / **添加文件夹**
2. 设置输出文件路径
3. 点击 **开始压缩**

程序自动选取密码加密，压缩完成后 ZIP 内所有文件名显示为 `xxxx.enc`。

### 解压

1. 选择 ZIP 文件
2. 设置输出目录
3. 点击 **开始解压**

程序自动遍历密码字典匹配，解压后文件名完整还原（含扩展名）。

## ❓ 常见问题

**Q: 加密后文件名能还原吗？**
A: 可以。使用 FuFu-zip 解压时，文件名会自动还原（包含原始扩展名）。

**Q: 忘记密码怎么办？**
A: 无需记忆密码。FuFu-zip 内置密码字典，会自动匹配解密。请确保使用相同版本。

**Q: 和普通 ZIP 加密有什么区别？**
A: 普通 ZIP 加密仅保护内容，文件名仍然可见。FuFu-zip 将文件名也一并加密，彻底隐藏文件信息。

**Q: 加密后的文件可以用其他软件打开吗？**
A: 不可以。只有 FuFu-zip 能解密，这也是保护隐私的核心机制。
